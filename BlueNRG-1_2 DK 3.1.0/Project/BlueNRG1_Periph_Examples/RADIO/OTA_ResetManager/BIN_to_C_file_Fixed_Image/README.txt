/* ****** How to create .c file for RADIO OTA Server FixedImage configuration ****** */
/* ********************************************************************* */

The script convert_bin2c_fixed_ota.exe generates a .c file to be used by the project example RADIO/OTA_ResetManager configuration OTA_Server_FixedImage. The script must be used as follows:

convert_bin2c_fixed_ota.exe -i file_name.bin -o file_name.c

So, for example, the following line shows as generate the .c file from the binary generated by the project RAIDO/TxRx configuration RX_Use_OTA_ResetManager:

convert_bin2c_fixed_ota.exe -i RADIO_RX_Use_OTA_ResetManager.bin -o RADIO_RX_Use_OTA_ResetManager.c

For much details to how create a binary file to be used for the RADIO OTA operations, see the User Manual UM2379 - BlueNRG-1, BlueNRG-2 Radio Driver.
The example RADIO/TxRx can be used as reference for generating such binary files, see the configuration RX_Use_OTA_ResetManager and/or TX_Use_OTA_ResetManager.
